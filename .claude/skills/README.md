# .claude/skills index

Canonical location for agent skills used by GitHub Copilot and Claude Code: `.claude/skills/`

Available skills:

- `git-rules` — Git/PR 運用ルールの SKILL と REFERENCE（`.claude/skills/git-rules/SKILL.md` と `REFERENCE.md`）。

編集方針:
- スキルの一次資料（REFERENCE.md）は `.claude/skills/<skill>/REFERENCE.md` に置くこと。
- SKILL.md はエージェントのための使用説明（いつ使うか、手順）を含めること。
- 更新は PR を通して行ってください。
